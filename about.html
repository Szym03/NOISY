About

Noisy is an evolving collection of generative sound experiences â€” from meditative sine compositions to richly textured noise, and interactive ambient environments.

Created with care using Csound and p5.js, each piece is designed to be calming, without becoming overly repetitive using algorithmic composition techniques. Currently, all code is written by me. If you have generative works that you would like to see on Noisy, feel free to reach out. You can also use any of the code on Noisy as you please. You can find it in this repository .

This project is intended for people who want sound in the background while working, resting, or sleeping. The goal is to create pieces that are interesting when listened to closely, but unobtrusive enough to fade into the background when needed.
